---
title: "Get Contact"
api: "GET https://api.clemta.com/contacts/{contact_id}"
description: "Retrieves a specific contact by its ID"
---

## Authentication

This endpoint requires authentication using an API key. Include your API key in the request header:

```
X-API-Key: your_api_key_here
```

See [Authentication](/api-reference/authentication) for more details.

## Path Parameters

<ParamField path="contact_id" type="string" required>
  The unique identifier of the contact to retrieve.
</ParamField>

## HTTP Response Codes

| Code | Description                                          |
| ---- | ---------------------------------------------------- |
| 200  | Contact retrieved successfully                       |
| 400  | Invalid request                                      |
| 401  | Unauthorized - Invalid or missing API key            |
| 404  | Contact not found                                    |
| 500  | Internal server error                                |

## Error Response

When an error occurs, the API returns an error response:

```json
{
  "success": false,
  "message": "Contact not found",
  "data": null,
  "error": {
    "code": "NOT_FOUND",
    "details": ""
  }
}
```

## Code Examples

<CodeGroup>

```bash cURL
curl -X GET https://api.clemta.com/contacts/64b8f1a2e4b0c8d9f0123456 \
  -H "X-API-Key: your_api_key_here"
```

```javascript JavaScript
const contactId = '64b8f1a2e4b0c8d9f0123456';

const response = await fetch(`https://api.clemta.com/contacts/${contactId}`, {
  method: 'GET',
  headers: {
    'X-API-Key': 'your_api_key_here'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

contact_id = "64b8f1a2e4b0c8d9f0123456"
url = f"https://api.clemta.com/contacts/{contact_id}"
headers = {
    "X-API-Key": "your_api_key_here"
}

response = requests.get(url, headers=headers)
print(response.json())
```

</CodeGroup>

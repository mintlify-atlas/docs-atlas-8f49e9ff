---
title: "Delete Contact"
api: "DELETE https://api.clemta.com/contacts/{contact_id}"
description: "Deletes a contact"
---

## Authentication

This endpoint requires authentication using an API key. Include your API key in the request header:

```
X-API-Key: your_api_key_here
```

See [Authentication](/api-reference/authentication) for more details.

## Path Parameters

<ParamField path="contact_id" type="string" required>
  The unique identifier of the contact to delete.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  Response message.
</ResponseField>

<ResponseField name="data" type="object">
  No data returned for delete operation (null).
</ResponseField>

## HTTP Response Codes

| Code | Description                                          |
| ---- | ---------------------------------------------------- |
| 200  | Contact deleted successfully                         |
| 400  | Invalid request                                      |
| 401  | Unauthorized - Invalid or missing API key            |
| 404  | Contact not found                                    |
| 500  | Internal server error                                |

## Error Response

When an error occurs, the API returns an error response:

```json
{
  "success": false,
  "message": "Contact not found",
  "data": null,
  "error": {
    "code": "NOT_FOUND",
    "details": ""
  }
}
```

## Code Examples

<CodeGroup>

```bash cURL
curl -X DELETE https://api.clemta.com/contacts/64b8f1a2e4b0c8d9f0123456 \
  -H "X-API-Key: your_api_key_here"
```

```javascript JavaScript
const contactId = '64b8f1a2e4b0c8d9f0123456';

const response = await fetch(`https://api.clemta.com/contacts/${contactId}`, {
  method: 'DELETE',
  headers: {
    'X-API-Key': 'your_api_key_here'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

contact_id = "64b8f1a2e4b0c8d9f0123456"
url = f"https://api.clemta.com/contacts/{contact_id}"
headers = {
    "X-API-Key": "your_api_key_here"
}

response = requests.delete(url, headers=headers)
print(response.json())
```

</CodeGroup>

## Example Response

```json
{
  "success": true,

```

## Important Notes

<Warning>
  Deleting a contact is a permanent action and cannot be undone. Make sure you want to delete the contact before proceeding.
</Warning>

<Info>
  When a contact is deleted, any references to this contact in other parts of the system may be affected. Consider archiving or deactivating contacts instead of deleting them if you need to maintain historical records.
</Info>

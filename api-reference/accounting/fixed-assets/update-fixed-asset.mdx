# Update Fixed Asset

This endpoint allows you to update an existing fixed asset.

## Headers

| Parameter    | Type   | Description                  |
| ------------ | ------ | ---------------------------- |
| X-API-KEY    | string | Formation API Key (required) |
| Content-Type | string | application/json             |

## Path Parameters

| Parameter       | Type   | Required | Description                   |
| --------------- | ------ | -------- | ----------------------------- |
| fixed_asset_id  | string | Yes      | Unique ID of the fixed asset  |

## Request Body

```json
{
  "title": "Updated Office Computer",
  "placed_in_service": 1640995200,
  "cost": 1800.00,
  "salvage_value": 200.00,
  "business_use": 100.0,
  "company_id": "64b8f1a2e4b0c8d9f0123456"
}
```

### Required Fields

| Field             | Type    | Description                                         |
| ----------------- | ------- | --------------------------------------------------- |
| title             | string  | Title/name of the fixed asset                       |
| placed_in_service | integer | Timestamp when the asset was placed in service      |
| cost              | number  | Original cost of the fixed asset                    |
| salvage_value     | number  | Estimated salvage value of the asset                |
| business_use      | number  | Percentage of business use (0-100)                  |
| company_id        | string  | ID of the company this fixed asset belongs to       |

### Field Descriptions

- **title**: A descriptive name for the asset (e.g., "Office Computer", "Company Vehicle", "Manufacturing Equipment")
- **placed_in_service**: Unix timestamp representing when the asset was put into use for business purposes
- **cost**: The original purchase price or acquisition cost of the asset
- **salvage_value**: The estimated residual value of the asset at the end of its useful life
- **business_use**: Percentage representing how much the asset is used for business (100 = 100% business use)
- **company_id**: The unique identifier of the company that owns this asset
